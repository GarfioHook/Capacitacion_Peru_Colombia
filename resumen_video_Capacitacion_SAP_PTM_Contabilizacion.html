Capacitacion SAP PTM - Contabilizacion

Reproductor Interactivo

Resumen Temático

Objetivo

Capacitar en los conceptos fundamentales de la contabilidad general (Record-to-Report) en SAP S/4HANA, abarcando la estructura contable, la visualización de documentos y la creación de asientos contables manuales.

Pasos del Proceso

  • Comprensión de la estructura contable: Ledgers (0L, LT), monedas (local, grupo, documento) y Plan de Cuentas.
  • Visualización de un documento contable existente utilizando la transacción FB03.
  • Análisis de la estructura de un documento: Cabecera y Posiciones.
  • Creación de un asiento contable manual con la transacción FB01.
  • Completar los datos de cabecera del documento (fechas, clase de documento, moneda).
  • Ingresar las posiciones del asiento utilizando Claves de Contabilización (40 para Debe, 50 para Haber).
  • Búsqueda y selección de cuentas de mayor, filtrando por sociedad.
  • Simulación del asiento para revisión previa a la contabilización.
  • Contabilización final del documento.
  • Análisis de la trazabilidad desde un documento logístico (Pedido de Compra) hasta el asiento contable automático generado.

Transacciones SAP

FB03FB01

Puntos Clave

  • La contabilidad en SAP es bimonetaria, manejando siempre moneda local (ML) y moneda de grupo, además de la moneda del documento.
  • El sistema utiliza Ledgers paralelos (ej. 0L para IFRS, LT para impuestos) para cumplir con diferentes normativas contables.
  • La mayoría de los asientos contables (aprox. 80%) se generan automáticamente desde otros módulos como MM (MIGO) o SD.
  • Para asientos manuales, se utiliza la transacción FB01 y clases de documento como 'AB' o 'SA'.
  • Las Claves de Contabilización son fundamentales: '40' para Débito y '50' para Crédito en asientos de cuenta de mayor.
  • Las cuentas del plan de cuentas corporativo (CAPJ) deben ser extendidas a una sociedad específica para poder ser utilizadas en ella.
  • Existe una completa trazabilidad (flujo de documentos) desde las operaciones logísticas hasta los registros financieros.
  • Las cuentas de gastos requieren una imputación a un objeto de controlling, como un Centro de Costo.

Recomendaciones

  • Para buscar cuentas contables, filtrar siempre por sociedad para ver solo las cuentas habilitadas y relevantes.
  • Utilizar la tecla F1 para obtener ayuda detallada sobre cualquier campo del sistema.
  • Utilizar la tecla F4 (matchcode) para buscar los valores posibles de un campo.
  • Antes de contabilizar, siempre usar la función de 'Resumen de documento' para simular y verificar el asiento.
  • Para la última línea de un asiento, se puede usar un asterisco (*) en el campo de importe para que el sistema calcule el saldo y cuadre el documento automáticamente.
  • Establecer un criterio uniforme a nivel de compañía para el uso de los campos de texto como 'Referencia' y 'Asignación' para facilitar análisis futuros.
