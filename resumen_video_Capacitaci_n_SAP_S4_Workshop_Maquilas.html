Guía Sesión Maquilas

Resumen Temático

Objetivo

Capacitar sobre el proceso de maquila, explicando cómo se gestiona en SAP S/4HANA a través del flujo de subcontratación (subcontracting) para transformar componentes en un producto terminado, ya sea con un proveedor externo o de forma interna.

Pasos del Proceso

  • Crear una Lista de Materiales (BOM) para definir los componentes del producto terminado (CS01).
  • Generar un Pedido de Compra de Subcontratación (ME21N) utilizando el tipo de posición 'L' para el producto final.
  • Registrar el envío de los componentes al proveedor mediante un traspaso de stock (MIGO, movimiento 541).
  • Supervisar el stock de componentes en poder del proveedor (ME2O).
  • Realizar la entrada de mercancías (MIGO) del producto terminado, lo que consume automáticamente los componentes del stock del proveedor y registra el costo del servicio.

Transacciones SAP

ME21NCS01CS03MMBEME2OMIGOMM02

Puntos Clave

  • El proceso de negocio 'Maquila' se implementa en SAP como 'Subcontratación' y pertenece al módulo de MM, no de PP.
  • La Lista de Materiales (BOM) es fundamental para que el sistema determine automáticamente los componentes necesarios.
  • En el pedido de compra (ME21N), el tipo de posición 'L' activa la funcionalidad de subcontratación.
  • El envío de componentes al proveedor es un traspaso de stock (movimiento 541) que no genera asiento contable, ya que la propiedad del material no cambia.
  • La recepción del producto terminado (MIGO) es el paso clave que da de alta el nuevo producto, consume los componentes y genera el impacto contable completo.
  • La transacción ME2O es la herramienta principal para monitorear y gestionar el inventario facilitado a los proveedores.

Recomendaciones

  • Definir correctamente la 'cantidad base' en la Lista de Materiales (BOM) para asegurar que las proporciones de los componentes sean correctas.
  • Utilizar 'alternativas' de BOM para gestionar diferentes versiones de componentes para un mismo producto terminado.
  • Para gestionar mermas o desperdicios de forma sistemática, se puede configurar un porcentaje de 'rechazo de componente' en la BOM.
  • Alinear internamente la definición de qué materiales (ej. tambores, etiquetas) son inventariables para asegurar su correcta inclusión en la BOM y en el costeo.
  • Para revertir un proceso, se debe anular el documento de material en MIGO, lo cual solo es posible si el stock resultante no ha sido movido o consumido posteriormente.
