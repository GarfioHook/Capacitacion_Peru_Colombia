Resumen Temático

Objetivo

Explicar y demostrar el proceso de expedición y facturación en SAP S/4HANA, que son los pasos posteriores a la creación de un pedido de venta en el ciclo Order-to-Cash (O2C).

Pasos del Proceso

  • Monitorear pedidos pendientes de entrega usando la transacción VL10A.
  • Crear la Entrega de Salida con referencia al pedido de venta (transacción VL01N).
  • Confirmar lote, almacén y cantidad en la entrega.
  • Contabilizar la Salida de Mercancía para registrar el costo de venta (COGS) y rebajar el inventario.
  • Crear el documento de facturación con referencia a la entrega de salida (transacción VF01).
  • Verificar el flujo de documentos completo para asegurar la trazabilidad del proceso.

Transacciones SAP

VA05VL10AVL01NVL02NVL03NVF01VF03MMBECKM3

Puntos Clave

  • El proceso O2C sigue un orden estricto: Pedido -> Expedición (Entrega) -> Facturación.
  • La transacción VL10A es el monitor central para gestionar las entregas pendientes, utilizando un sistema de semáforos.
  • La Salida de Mercancía (contabilizada desde la entrega) es el evento que mueve el inventario y genera el asiento de costo de venta (COGS).
  • La facturación (VF01) se crea con referencia a la Entrega de Salida, no al Pedido de Venta, y genera el asiento de ingreso y la cuenta por cobrar.
  • El Flujo de Documentos es una herramienta esencial para visualizar la relación y el estado de todos los documentos del ciclo (pedido, entrega, documento de material, factura).

Recomendaciones

  • Utilizar la transacción VL10A para la gestión operativa de las entregas en lugar de la VA05.
  • Asegurarse de que los datos maestros del material y cliente estén completos (ej. puesto de expedición, centro de beneficio) para evitar errores.
  • Verificar siempre la 'fecha de salida de mercancía real' en la entrega para garantizar la correcta contabilización del costo, especialmente en cierres de mes.
  • Para analizar listados, aprovechar las funcionalidades de totalizar y subtotalizar para agrupar la información de manera eficiente.
